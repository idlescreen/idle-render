# render

Offline video rendering engine for IdleScreen.

🌐 **Website & Installation:** [https://idlescreen.github.io](https://idlescreen.github.io)

### CLI Commands

```bash
render --effect beams --duration 10s -o /tmp/beams.mkv
render --effect storm --duration 8h --segment 1h --resolution 4k -o /tmp/night.mkv
```
